# cls file

## 1.cls文件是什么
latex 文档的默认格式由使用的`calss`决定

`class` 文件后缀名 `.cls`

`package` 文件后缀名为 `.sty`


## class 和 package 的不同

基本原则是，如果文件包含控制一种特殊类型文档外观的逻辑结构的一些命令，那么就用`calss`

否则，如果仅仅只是加一些文档类型之外的一些 feature 到你的文件中，那么就用 `package`
